Image Crawler Completable Future App (Assignment 3)

The purpose of this assignment is to deepen your experience using Java's functional programming features and their integration with Java's completable futures framework. In particular, in this assignment you'll develop an app that uses Java's completable futures framework in conjunction with Java's sequential streams framework to obtain, store, and transform images.

Like the earlier assignments, the code you'll write here will have a GUI and will run as an Android app. Please use this GUI along with the unit tests outlined below to help test and debug your solution.

Assignment Structure and Functionality

The ImageCrawler app is packaged as a project using the latest version of Android Studio. This app is written in both Kotlin and Java and demonstrates many Android capabilities. For the purposes of Assignment 3, however, you only need to be aware of the following directories:

  • image-crawler/src/main/java/edu/vanderbilt/imagecrawler/crawlers, which contains the skeleton(s) you'll need to fill in, as described below.
  • image-crawler/src/test, which contains a set of unit tests that exercise the features you're implementing and are used to evaluate whether you've correctly satisfied this assignment's requirements.
  • app/src/androidTest/java/assignmentTests, which contains an Android Studio instrumentation test that runs your app automatically.

To compile this code you need to use the provided Android Studio project. You can run this project by clicking the green "Run 'app'" button in the Android Studio IDE, which should automatically select an Android Emulator to run, assuming you have one created. The application's minimum API is 29, so you will need to select an emulator that uses an API >= 29. If you do not already have one created you can create it by clicking on the "AVD Manager" button in the Android Studio IDE.

Program Description and "TODO" Tasks

In this assignment you'll complete the implementation of an image crawler class in the CompletableFutureCrawler.java file that obtains, transforms, and stores images. This class uses the Java completable futures framework to perform an "image crawl" asynchronously, starting from a root Uri. Depending on the parameters used to run the tests, images from the HTML page reachable from the root Uri are either downloaded from a remote web server or read from the local file system and the results are stored in files.

The bulk of the code for the app is provided in the skeletons. In fact, a complete solution using Java 7 features is provided in the SequentialLoopsCrawler.java file. Your will therefore convert this Java 7-based class to the CompletableFutureCrawler class that uses Java completable futures.

Your solution should use no loops or (unnecessary) conditional statements in this assignment and instead use Java's CompletableFuture methods and Java sequential streams.
